Mission Statement

Spotifoo puts music back in the hands of the user. Our crowsourced API collects and aggregates music artists, songs, and albums. The API also allows for the curation of playlists. Once you're signed up you can add songs directly to the database and construct playlists with the available music. Through crowdsourcing information, we at Spotifoo strive to build a database where all those with any musical inclination can share music that makes them want to belt out as if they were alone atop a mountain.

Route Testing

/users/

  • /users/ gets all users currently signed up in our database

/users/id

  • /users/id/ returns a particular user by their id

/songs/

  • /songs/ route is our smallest point of entry.
  • /songs/ will get you an array object of all songs currently listed in Spotifoo

/songs/id

  • /songs/id/ get a select song with it's mongodb generated id.

/artists/

  • /artists/ route gets all artists in Spotifoo's database including albums and genre

/artists/id

  • /artists/id/ specific artist by id.

/albums/

  • /albums/ shows all albums in our database, including song tracks and song length.

/albums/id

  • /albums/id/ shows specific album with matching Spotifoo id.

/playlists/

  • the playlists route gets all playlists

/playlists/id

  • the playlists id route get playlist by Id and to delte playlist

/plSongId/

  • route allows for posting and deletion of playlist songs by their ID.

/auth/

  • our authentication wrote houses our /signup /signin and /verify routes which autheticate users
